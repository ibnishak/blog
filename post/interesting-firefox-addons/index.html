Interesting Firefox Addons and Other things

September 15, 2022    |     Reading Time: 1 min 3 s  
  1. Tagspaces Web Clipper: This Add-on will help you to save web pages or parts of them as files, for later reading or organizing with app such as TagSpaces’ desktop.
  2. Webscrapbook: Archive webpages offline, organize and highlight text in webpages.
  3. Single-Filez: Better sibling of Single File Addon. Webpage is saved as a self extracting zip file and webpage resources like fonts, images, stylesheets etc are saved as separate files. Nevertheless, you can view webpage in your browser without extracting - coz magic.
  4. Watchmarker for Youtube: By default Youtube marks your recently viewed videos as watched. This extension marks videos watched in your entire youtube history.
  5. Bebop: Emacs helm like interface for Firefox
  6. Bloody Vikings: Instead of exposing your real email address to every random site asking for it, simply right-click on the designated input field and select “Bloody Vikings!”. Bloody Vikings! automatically inserts a temporary email address and opens the corresponding mailbox in a new background tab.
  7. UI Vision RPA: Add macros and UI automation to firefox
  8. Find On Reddit: Find reddit threads about current URL
  9. What hackernews says: Find hackernews threads about current url.
  10. Humble new tab page: Have new tab page show your bookmarks.

Other Popular Addons

  1. Ublock Origin with Fanboy Lists to enhance the power to uBlock origin
  2. uBlacklist: Blacklist websites from appearing in google search results.
  3. Temporary Containers: Disposable containers that isolate attached data and are deleted after usage. #stack
