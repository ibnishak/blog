Login Without Display Manager

April 30, 2023    |     Reading Time: 5 s  

Add the following to /etc/zsh/zprofile

if [[ -z "$DISPLAY" ]] && [[ $(tty) = /dev/tty1 ]]; then
	exec startx
fi
